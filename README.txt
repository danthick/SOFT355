To open the application please run the following commands in a terminal:
npm install
node server


This will install the required dependencies and start the server.
Navigate to http://localhost:9000/ to access the application.





To compile the unit tests run:
mocha test --exit



Jenkins SSH connection:
ssh -L 127.0.0.1:8080:localhost:8080 danthick@soft355-jenkins.uksouth.cloudapp.azure.com
